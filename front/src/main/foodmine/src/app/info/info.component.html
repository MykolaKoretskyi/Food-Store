About the program:

The program is a demonstration version of an online food ordering store. Please note that the interface language is English, the search is carried out by name in English.

For testing purposes, you can register as a customer. You will be granted the 'USER' role, and you can also test an already registered user with administrative privileges for order management with the 'ADMIN' role, using the following credentials: Username: 'Manager_1', Password: 'Manager_1'.

Technical information:

The following technologies were used to develop the program:

  • Angular 16
  • Java 17
  • Maven
  • Spring 3.0.6
  • Spring Data
  • Hibernate
  • Spring MVC (REST)
  • Spring Security JWT
  • PostgreSQL
  • Lombok
  • Amazon RDS
  • Amazon S3
  • AWS Elastic Beanstalk
